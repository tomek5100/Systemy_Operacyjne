Zadanie 2. Algorytm piekarni i licznik

Zaimplementować algorytm piekarni przy pomocy wątków POSIX. Poprawność działania programu zademonstrować na podstawie globalnego licznika. Oprócz wartości końcowej licznika, wątki nie mają nic wyświetlać. Nie należy korzystać z innych mechanizmów synchronizacji (muteksów, semaforów, kolejek komunikatów, etc.).

Program powinien działać według poniższego pseudokodu:

volatile int licznik = 0 // Globalny licznik

thread() { // Funkcja wątku

    for(i = 1, ..., 1000000) {
        lock()  // Wejście do sekcji krytycznej
                // według algorytmu piekarni

        licznik = licznik + 1

        unlock() // Wyjście z s.k.
    }
}

Należy stworzyć kilka wątków (np. 4), które jednocześnie zwiększają licznik, a po ich zakończeniu sprawdzić czy licznik ma poprawną wartość.

Sprawdzić wynik po usunięciu wywołań funkcji lock() i unlock().
